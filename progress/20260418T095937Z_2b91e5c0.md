## Dependency DAG

```
Fse/Basic (foundation)
├── Fse/ProbsGeNeg1
├── Fse/Distribution → Fse/BitPos

└── Fse/BuildTable → Fse/Predefined
Fse/BackwardBitReader (standalone, imports Zstd.Native.Fse directly)
```

## Decisions made

- **`Except.bind_eq_ok'` promoted to public**: was `private` in the
  original but needed by both `BuildTable` and `Predefined`. Moved to
  `Basic.lean` as a public helper.
- **BuildTable at 755 lines**: still below 800, and splitting it further
  would require bi-directional lemma movement between sub-files (the
  accuracyLog/cells_size/numBits_le chain feeds into buildFseTable_ok
  which feeds into newState_lt / symbol_lt / valid). Kept together.
- **Distribution does not depend on ProbsGeNeg1**: ProbsGeNeg1 is a
  leaf that consumers can import if they need probability-bounds
  guarantees, but the Distribution correctness theorems themselves
  don't use them.
