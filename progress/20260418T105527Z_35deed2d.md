# 2026-04-18 10:55 UTC — Feature: Split Spec/Huffman.lean

Session: `/work` → `/feature` for issue #13

## Accomplished

Split `Zstd/Spec/Huffman.lean` (1822 lines) into six thematic submodules
under `Zstd/Spec/Huffman/`, each well under the ~800-line target. The
top-level `Zstd/Spec/Huffman.lean` is now a 15-line re-export shim, so
existing imports (`import Zstd.Spec.Huffman`) continue to work.

| Submodule          | Lines | Content                                                              |
|--------------------|-------|----------------------------------------------------------------------|
| `Weights.lean`     |   138 | Weight predicates: `weightSum`, `ValidWeights`, `isPow2`, `KraftComplete`, bitwise helpers |
| `Table.lean`       |   492 | `ValidHuffmanTable`, `findMaxBitsWF_*`, `fillHuffmanTableWF_*`, `buildZstdHuffmanTable_*` |
| `LiteralsSection.lean` | 418 | `parseCompressedLiteralsHeader_*`, `parseLiteralsSection_*` structure/size/content lemmas |
| `TreeDescriptor.lean`  |   193 | `parseHuffmanWeights{Direct,Fse}_*`, `parseHuffmanTreeDescriptor_*` bounds/validity |
| `Decode.lean`      |   258 | `decodeHuffmanSymbol_*`, `decodeHuffmanStreamWF_*`, `decodeHuffmanLiterals_succeeds_*` |
| `Completeness.lean`|   413 | `*_succeeds` parsing completeness theorems + `rawLiteralsSectionSize`, `rleLiteralsSectionMinSize` |

Import graph:
- `Weights` — no Zstd imports (self-contained predicates)
- `Table` — imports `Weights` + `Native.Huffman`
- `LiteralsSection` — imports `Native.Huffman` only
- `TreeDescriptor` — imports `Table` + `Native.Huffman`
- `Decode` — imports `Table` + `Spec.Fse` + `Native.Huffman`
- `Completeness` — imports `TreeDescriptor` + `Spec.Fse` + `Native.Huffman`

## Decisions

- Kept `parseCompressedLiteralsHeader` and `parseLiteralsSection` in the
  same module because the private helper
  `parseCompressedLiteralsHeader_headerBytes_ge` is needed by
  `parseLiteralsSection_compressed_spec`; splitting would have required
  exporting it.
- `Decode.lean` needed both `Zstd.Spec.Fse` (for `readBits_*` lemmas) and
  `Zstd.Native.Fse` (transitively) for `BackwardBitReader`.
